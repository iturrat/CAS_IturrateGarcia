# Module 4 Project Overview

This README file describes the main aim of the project for Module 4.

## Project Assignments

For this project, the assignments are as follows:

- Create your own GitHub repository for your CAS material and projects.
- Document the repository and subfolders (one folder per module, one for the final project) with README files.
- **Optional:**
  - Create a GitHub webpage for your CAS repository and/or
  - Document your Module 3 scripts with Sphinx (or another tool) and publish it on ReadTheDocs.
- Add the link to your GitHub repository on the Module 4 Ilias.
