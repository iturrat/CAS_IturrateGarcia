This README file describes the main aim of the project for Module 4. 

For this project the assignments are:

- Create your own GitHub repository for your CAS material and projects 
- Document repository and subfolders (one folder per module, one for the final project) with Readme files
- Optional: 
	Create a GitHub webpage for your CAS repository or/and 
	Document your Module 3 scripts with sphinx (or some other tool) and publish it on ReadTheDocs
- Add the link to your GitHub repository on the Module 4 Ilias
